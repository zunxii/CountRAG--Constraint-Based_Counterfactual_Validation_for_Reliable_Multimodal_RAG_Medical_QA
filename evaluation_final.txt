EVALUATION
This section presents a comprehensive experimental evaluation of the proposed Research RAG system. The evaluation is designed to assess three key aspects of the system: retrieval effectiveness, component-level contribution, and robustness under modality perturbations. All experiments are conducted to validate the impact of domain adaptation, multimodal fusion, and concept-based retrieval on diagnostic performance.

A. Evaluation Methodology
We evaluate the retrieval performance of the system using standard Information Retrieval (IR) metrics commonly employed in multimodal and medical retrieval settings.

Recall@K (R@K).
Recall@K measures the proportion of queries for which at least one relevant case appears within the top K retrieved results. We report R@1, R@5, and R@10 to capture both strict and relaxed retrieval settings. R@1 reflects the system’s ability to place the correct diagnosis at the top rank, while higher values of K evaluate broader retrieval coverage.

Mean Reciprocal Rank (MRR).
MRR evaluates the ranking quality of retrieval results by measuring the reciprocal rank of the first relevant result. This metric is particularly important in clinical settings, where presenting a correct diagnosis early in the ranked list is critical.

Jensen–Shannon Divergence (JSD).
To evaluate robustness, we compute the Jensen–Shannon Divergence between the original prediction distribution and distributions obtained under counterfactual perturbations (e.g., removing image or text inputs). Lower JSD values indicate greater stability and balanced modality reliance.

B. Experimental Setup
The system is evaluated using 587 queries from the held-out test split. Each query consists of a clinical text description and, where available, a corresponding medical image.

Modes Evaluated
To analyze the contribution of each modality, we evaluate the system under three retrieval modes:
Text Mode: Retrieval performed using only clinical text embeddings.
Image Mode: Retrieval performed using only medical image embeddings.
Fusion Mode: Retrieval using the proposed adaptive image–text fusion mechanism.

All modes share the same retrieval backend and ranking procedure to ensure a fair comparison.

The results show that Fusion Mode consistently outperforms both unimodal baselines across all metrics. While Image Mode already provides strong performance, incorporating textual information through adaptive fusion yields a clear improvement, particularly in top-rank accuracy (R@1). This confirms that multimodal integration provides complementary diagnostic cues.

To further understand system behavior, we analyze retrieval performance on a subset of diagnostically challenging cases. Table II reports R@1 scores for selected diagnoses.

D. Modality Contribution Analysis
The strong performance of Fusion Mode (R@1 = 0.81) compared to Text Mode (0.59) and Image Mode (0.75) indicates improved robustness to modality-specific limitations. The improvement observed for Edema (Fusion: 0.85 vs. 0.50 for both unimodal modes) highlights the system’s ability to leverage complementary information from both modalities.

E. Counterfactual Stability Analysis
To rigorously assess the system's reliance on individual modalities, we performed a counterfactual analysis by ablating text or image inputs and measuring the Jensen-Shannon Divergence (JSD) from the baseline prediction distribution. Lower JSD implies high stability (the system's prediction did not change significantly when a modality was removed).

Table III summarizes the average JSD scores across the test set (N=587). Detailed stability profiles for key diagnoses are presented in Table IV.

TABLE III
COUNTERFACTUAL STABILITY SCORES (Lower is Better)
Perturbation Type     Mean JSD    Interpretation
No Text (Image Only)  0.1214      High Stability (Low Reliance on Text)
No Image (Text Only)  0.2676      Moderate Stability (High Reliance on Image)
Noisy Input           0.0430      Very High Stability

The results indicate that the system is generally more robust to missing text (JSD=0.12) than missing images (JSD=0.27). This aligns with the findings in Table I where Image Mode outperformed Text Mode. However, the non-zero JSD for 'No Text' confirms that text still plays a role in refining the prediction distribution.

We further categorized each query into robustness levels based on their max JSD score.
- High Robustness (JSD < 0.1): 40.89% (240 samples)
- Medium Robustness (0.1 < JSD < 0.4): 9.54% (56 samples)
- Low Robustness (JSD > 0.4): 49.57% (291 samples)

Notable observations:
1. Neck Swelling exhibits exceptional robustness, with near-zero JSD for both perturbations, suggesting the system has learned redundant, high-confidence features for this condition.
2. Edema and Swollen Tonsils show high sensitivity to image removal (JSD > 0.45), indicating that for these conditions, visual features are the primary diagnostic driver, and text descriptions alone are often insufficient for precise retrieval.

Conclusion
The counterfactual analysis confirms that while the Fusion mechanism improves overall accuracy, the system maintains a strong visual bias, which is appropriate for a dermatology/visual-focused diagnostic task. However, the adaptive capability allows it to fall back effectively when one modality is noisy or less informative.
